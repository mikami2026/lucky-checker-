# Lucky Checker 🍀

## アプリ概要
自分がどれだけ恵まれた環境にいるかを客観的に知ることができるアプリです。

水道水・電気・医療など10項目について、
4つの選択肢から自分の状況に近いものを選ぶと、
世界基準で見た自分の豊かさが100点満点で表示されます。（ただし、最高スコアは80点となっております。）

## デモ
https://mikami2026.github.io/lucky-checker-/

## 使い方
1. 10個の質問に答える
2. 各質問で自分の状況に最も近い選択肢を選ぶ
3. 100点満点で結果が表示される

画面右上のボタンで日本語/英語を切り替えられます。

## 年収くらべ機能（日本国内版）
トップ画面の「💴 年収くらべ」から年収（万円・額面）を入力すると、
日本の公的統計と比較した結果が3つ表示されます。

1. **給与所得者の中での位置（上位◯%）**
   国税庁「令和5年分 民間給与実態統計調査」より
2. **同年収層の貯蓄額（中央値・平均値）**
   金融広報中央委員会「家計の金融行動に関する世論調査 2023年」（二人以上世帯）より
3. **同年収層の消費支出（月額）**
   総務省統計局「家計調査 2023年」（二人以上世帯）より

※貯蓄額は世帯がこれまでに貯めてきた金融資産の合計（ストック）であり、
1年間に貯めた額ではないため、年収より多くなることがあります。
