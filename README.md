# Lucky Checker 🍀

## アプリ概要
自分がどれだけ恵まれた環境にいるかを客観的に知ることができるアプリです。

水道水・電気・医療など10項目について、
4つの選択肢から自分の状況に近いものを選ぶと、
世界基準で見た自分の豊かさが100点満点で表示されます。（ただし、最高スコアは80点となっております。）

## デモ
https://0gr39007726117p-netizen.github.io/lucky-checker-/lucky-checker.html

## 使い方
1. 10個の質問に答える
2. 各質問で自分の状況に最も近い選択肢を選ぶ
3. 100点満点で結果が表示される

## 使用技術
- HTML

## 作成背景
世界には様々な生活環境の人がいる中で、
自分の生活を客観的に見つめ直すきっかけを
作りたいと思い開発しました。
