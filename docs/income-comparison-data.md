# 年収比較機能用 統計データ（2026-07-14 調査）

年収を入力すると、同年収層の位置づけ・貯蓄額・消費支出を比較表示する機能のために調べた公的統計データ。
実装時にこのファイルの数値をそのまま使用する想定。

## 実装メモ

- 質問構成は「年収を数値入力 → 上記データから自動計算して3つの比較結果を表示」（ユーザーと合意済み）
- 既存の `lucky-checker.html` の質問カードと同じUI（アイコン・アニメーション・カード形式）を踏襲する
- 表1で百分位（上位◯%）、表2で貯蓄額の位置づけ、表3で消費水準の位置づけを算出する
